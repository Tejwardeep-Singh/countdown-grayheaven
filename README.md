<h1 align="center">Countdown - Grayheaven</h1>

## 🎮 Overview

**Countdown: Gray Heaven** is a third-person narrative thriller developed in **Unity**.
The game follows detective **Tim Crowe**, who must uncover hidden clues across a mysterious environment before a devastating event unfolds.

The player explores a dark and atmospheric setting, interacts with audio clues, and pieces together a story that gradually reveals the truth behind a hidden conspiracy.

---

## 🧩 Features

* Third-person exploration gameplay
* Interactive clue system
* Story-driven progression
* Atmospheric environment and sound design

---

## 🕹 Gameplay

The player investigates different objects in the environment to unlock story clues.

Key mechanics include:

* **Inspect system** for interacting with objects
* **Audio clue system** that drives story progression
* **Dynamic UI prompts**
* **Story manager system** controlling narrative flow

## 🚀 How to Run

### Unity Editor

1. Clone the repository
2. Open the project in **Unity Hub**
3. Open the main scene
4. Press **Play**

### Build Version

Download the latest build from the **Releases** section or from itch.io.

---

## 🎮 Controls

### PC

```
W A S D  → Move
Mouse    → Look around
E        → Interact / Inspect
```

### Mobile

```
Left Side   → Movement
Right Side  → Camera look
Inspect     → Interaction
```
